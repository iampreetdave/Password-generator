# Password Generator

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

> A simple and secure random password generator.

**This project is currently under development.**

## About

Password Generator is a utility tool designed to create strong, random passwords with customizable length and character sets. Built to help users generate secure passwords quickly for personal or professional use.

## Tech Stack

- **Language:** Python

## Features

- Generate random passwords of any length
- Include/exclude uppercase, lowercase, numbers, and special characters
- Simple command-line interface

## Getting Started

### Prerequisites

- Python 3.7+

### Installation

```bash
git clone https://github.com/iampreetdave/Password-generator.git
cd Password-generator
```

### Run

```bash
python password_generator.py
```

## How It Works

The generator uses Python's `random` and `string` modules to create cryptographically varied passwords by sampling from configurable character pools (letters, digits, symbols) and assembling them to the desired length.
